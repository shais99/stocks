# Prerequisites

- MongoDB
- Node
- NPM

## Steps

- Pull the repository
- Run `npm i` in both `/client` and `/server`
- Run MongoDB (e.g.: `mongod --dbpath C:/dbpath`)
- Update the `.env` file accordingly

## Run the client

- Run `npm start` in the root (`/client`) directory

## Run the server

- Run `npm run start:dev` in the root (`/server`) directory
